# Trail

## Objetivo:
Desenvolver um APP para celular, para registo e consulta de trilhas.
## Plataforma: 
Android ( e IOS)
## Funcionalidades:
As funcionalidades são associadas às API do Mapas ad google, para poder sinalizar os pontos da trilha, num mapa.
#### 1.	Registo de uma trilha
•	Nome da Trilha
o	Localização ( País, Cidade ou Região)
o	Comprimento da trilha
o	Ponto no mapa, que é um sinalizador do percurso (identificação de um local da trilha)
	Campo a identificar a área ( parque estacionamento, banheiro, Obstáculo difícil)
#### 2.	Consulta de trilhas já cadastradas
	Listar as trilhas existentes, por seleção coloca-la no mapa do google, com identificação dos apontadores de sinalização do percurso.
